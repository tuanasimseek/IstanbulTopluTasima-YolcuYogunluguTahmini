# İstanbul Toplu Taşıma — Yolcu Yoğunluğu Tahmini

İBB'nin açık veri portalından alınan gerçek İstanbul toplu taşıma verisiyle,  
farklı makine öğrenmesi algoritmalarının karşılaştırmalı analizi yapıldı.

##  Proje Amacı
İstanbul'daki toplu taşıma hatlarında saatlik yolcu yoğunluğunu tahmin etmek.

##  Benim Katkım
Bu grup projesinde ben şu modelleri geliştirdim:
- **XGBoost** ile gradient boosting modeli
- **PyTorch** ile derin öğrenme yaklaşımı  
- **Linear Models** (Ridge, Lasso) ile baseline karşılaştırması
- Veri görselleştirme grafikleri

##  Kullanılan Teknolojiler
- Python, Jupyter Notebook
- scikit-learn, XGBoost, PyTorch
- Pandas, NumPy, Matplotlib

##  Ekip
5 kişilik grup projesi — her üye farklı ML algoritmaları uyguladı.
